# Combat Game — Cliente

Cliente del juego de combate multijugador por sockets TCP.

---

## Autor

**Yulian Alexis Tobar Rios**

Código: 202222448

---

## Tecnologías utilizadas

* Java 17
* Maven
* Arquitectura MVP (Model-View-Presenter)
* Interfaz gráfica con Java Swing
* Comunicación por sockets TCP
* Concurrencia con hilos (Thread / Runnable)
* Serialización JSON con Gson
* FlatLaf (Look & Feel moderno)
* Librería externa de componentes (components-lib)

---

## Requisitos

* Java 17 o superior
* Maven

---

## Dependencias

* **flatlaf 3.7.1**
* **gson 2.13.1**
* **components-lib 1.0.0**
